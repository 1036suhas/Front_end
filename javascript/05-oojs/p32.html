STATIC METHOD..!

JS provides static method that belongs to the class instead of an instance of an that class . these methods are called directly on the class itself.

POINTS to remainder

  • The Static keyword is used to declare a static method.
  • The Static method can be any name.
  • A class can contain more than one static method.
  • If we declare more than one static method with a similar name, the js always invokes the last one.
  • we can use this keyword to call astatic method within another static method.
  • we can't use this keyword directly to call a static method within non static method .
  • in such case we can call the static method either using clas name or as the property of the constroctor
